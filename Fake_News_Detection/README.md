# Fake News Detection System

## Project Overview
This project detects whether a news article is real or fake using machine learning and NLP techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Jupyter Notebook

## Project Workflow
1. Data exploration
2. Text preprocessing
3. Feature extraction
4. Model training and evaluation

## Model Performance
The trained model achieved an accuracy of around 99.11% on the test dataset.

## Future Improvements
- Improve model accuracy using deep learning
- Deploy the model as a web application
- Add real-time news prediction
